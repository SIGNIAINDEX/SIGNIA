# Signia Console (Web)

A Next.js console for SIGNIA:
- Compile structures
- Verify proofs
- Browse artifacts
- View registry status
- Ask the Interface service for deployment and project guidance

## Env
Create `.env.local`:

- `NEXT_PUBLIC_SIGNIA_API=http://localhost:8080`
- `NEXT_PUBLIC_SIGNIA_INTERFACE=http://localhost:7070`

## Run
```bash
npm install
npm run dev
```
